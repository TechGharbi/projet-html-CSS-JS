1) Les Composants matériel d’un Réseau informatique ce composent de :

• Les cartes réseau :sont des composants matériels qui permettent à un ordinateur de se connecter à un réseau, tel que Internet. Ces cartes sont installées à l'intérieur de l'ordinateur. Les cartes réseau jouent un rôle dans la communication entre ordinateurs sur un réseau. Elles permettent à l'ordinateur de transmettre et de recevoir des données via des protocoles de communication spécifiques, tels que TCP/IP.

• SWITCH : est utilisé pour connecter des appareils sur un réseau local . Il permet à plusieurs appareils de communiquer entre eux en utilisant des adresses MAC (Media Access Control). Lorsqu'un appareil se connecte au switch, il vérifie son adresse MAC et stocke cette information dans une table de correspondance.

• HUB : est utilisé pour connecter plusieurs appareils sur un (LAN). Il fonctionne comme un répéteur, Contrairement à un switch qui utilise une table de correspondance, Cela signifie que lorsque des données sont envoyées sur le réseau, tous les appareils connectés reçoivent ces données. Les hubs ont été remplacés par des switches, qui sont plus efficaces pour la gestion du trafic réseau et permettent une communication plus rapide , mais ils sont encore utilises ou l’utilisation d’un switch n’est pas nécessaire.

2)Le modèle OSI(Open System Interconnexion) :

est un modèle de référence pour les communications entre ordinateurs. Il est composé de sept couches, chacune ayant un rôle spécifique dans la transmission des données. La couche physique est responsable de la transmission des signaux physiques, la couche liaison de données gère la transmission des données entre les équipements, la couche réseau assure la transmission des paquets de données entre les réseaux, la couche transport garantit la transmission fiable des données, la couche session établit et gère les sessions de communication, la couche présentation s'occupe de la représentation des données et la couche application fournit des services aux applications. Le modèle OSI est important car il permet une communication standardisée entre les équipements informatiques, indépendamment de leur fabricant ou de leur technologie.

3) Le Modèle TCP/IP (Transmission Control Protocol/Internet Protocol) :

est un modèle de Référence pour les communications de réseau, qui est utilisé pour la conception et la mise en Œuvre des réseaux informatiques modernes. Contrairement au modèle OSI, le modèle TCP/IP divise les communications en quatre couches principales. La couche liaison de données qui est responsable de la transmission des données sur le support physique. La couche réseau qui est utilisé pour le routage des données à travers le réseau. La couche transport qui est responsable de l'acheminement des données des appareils. Bout en bout entre les appareils. La couche application cette couche est utilisé pour la communication entre les applications sur les appareils de réseau. Ce modèle est le modèle de référence pour les protocoles de communication de réseau les plus largement utilisés, tels que le protocole Internet (IP), Il est utilisé pour la conception et la mise en œuvre des réseaux d'entreprise, des réseaux Internet et des réseaux sans fil.

4)L’adressage :

Une adresse IP (Internet Protocol) est le numéro qui identifie chaque ordinateur dans les réseaux qu’il appartient. Il y’a deux mode d’adressage IPV4 et IPV6.
IPV4 a différents types d’adresse, L’adresse réseau : tous les bits de la partie hôtes=0.
L’adresse de diffusion : il s’agit d’une adresse dans laquelle tous les bits de partie hôtes=1.Pour le réseau 10.0.0.0 l’adresse de diffusion = 10.255.255.255. Cette adresse permette à un pc d’envoyer des paquets à l’adresse de diffusion de réseau. L’adresse hôtes : On attribue les valeurs situées entre l’adresse réseau et l’adresse de Diffusion sure {machines} situé sur ce réseau. Le masque de réseau et de sous réseaux : Tous les bits de la partie réseaux = 1 .
